frb.dm_kde
==========

This module family contains KDE and simulation utilities used in DM modeling.

.. warning::
   ``frb.dm_kde`` currently has no package ``__init__.py``. Import submodules
   directly (for example, ``import frb.dm_kde.dm_frb_sim``).

.. note::
   Optional dependency: ``asymmetric_kde`` is required for parts of the DM KDE workflow.
